# AI ChatBot

## Project Description
AI ChatBot is a Java Swing based desktop application developed for the CodeAlpha Java Programming Internship. It allows users to chat with a simple rule-based chatbot through a graphical user interface.

## Features
- Java Swing GUI
- Send Message
- Clear Chat
- Rule-Based AI Responses
- Date and Time Response
- Auto Scroll
- Enter Key Support
- User-Friendly Interface

## Technologies Used
- Java
- Java Swing
- OOP Concepts

## How to Run
1. Open the project in VS Code.
2. Compile the program:
   javac ChatBotGUI.java
3. Run the program:
   java ChatBotGUI
