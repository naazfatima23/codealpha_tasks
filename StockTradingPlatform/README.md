# Stock Trading Platform

## Description
The Stock Trading Platform is a Java Swing desktop application developed for the CodeAlpha Java Programming Internship. It allows users to buy, sell, update, delete, and manage stock records using a MySQL database.

## Features
- Buy Stock
- Sell Stock
- Update Stock Details
- Delete Stock Record
- View Stock Records in JTable
- Automatic Total Price Calculation
- MySQL Database Connectivity
- User-Friendly GUI

## Technologies Used
- Java
- Java Swing
- JDBC
- MySQL
- XAMPP

## Database
Database Name: stockdb

Table Name: stocks

## How to Run
1. Start Apache and MySQL in XAMPP.
2. Import the stockdb.sql file into phpMyAdmin.
3. Add mysql-connector-j.jar to your project.
4. Run the StockTradingPlatform.java file.
