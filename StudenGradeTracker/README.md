# Student Grade Tracker

## Description
Student Grade Tracker is a Java Swing application that helps users manage student marks. It allows adding student details, calculating average marks, highest marks, and lowest marks through a simple graphical user interface.

## Features
- Add Student Name
- Add Student Marks
- Display Student List
- Calculate Average Marks
- Find Highest Marks
- Find Lowest Marks
- Clear All Records

## Technologies Used
- Java
- Java Swing
- OOP
- ArrayList
